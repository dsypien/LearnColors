# Learn Colors
Learn Colors is a web application that I wrote for my daughter Emma, to help 
her learn colors. 

Dependencies
====

- 'git' to clone repository
- 'npm' to install grunt & grunt tools 

Building
====

For OSX/Linux:

```bash
$ git clone https://github.com/dsypien/LearnColors.git
$ cd LearnColors
$ npm update
$ grunt
```
